# upsgi 0.1.0 release notes

## Summary
upsgi 0.1.0 is the first public release of the PSGI-first fork.

## Highlights
- default build now produces the `upsgi` binary
- public version string is now `0.1.0`
- default embedded bundle remains limited to:
  - `psgi`
  - `logfile`
  - `logsocket`
  - `rsyslog`
- compatibility-only flags remain parseable and inert:
  - `http-modifier*`
  - `http-socket-modifier*`
  - `https-socket-modifier*`
  - `perl-no-die-catch`
- PSGI exception visibility remains opt-in via `--log-exceptions`
- static-map remains baseline through the core request/static path
