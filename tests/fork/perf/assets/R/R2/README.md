# R2 asset bundle

Compressed ingest fixture with in-memory gzip inflation and append-path verification.

Contents:
- shared receiver app in `../common/receiver_app.psgi`
- shared corpus and client helpers in `../common/`
- `config.yaml.in` for the scenario
- `r2_matrix.json` for the scenario matrix
- `run_r2_smoke.sh` lightweight smoke runner
