# R5 asset bundle

Append-pressure fixture with optional artificial queue delay to make sink pressure visible during smoke runs.

Contents:
- shared receiver app in `../common/receiver_app.psgi`
- shared corpus and client helpers in `../common/`
- `config.yaml.in` for the scenario
- `r5_matrix.json` for the scenario matrix
- `run_r5_smoke.sh` lightweight smoke runner
