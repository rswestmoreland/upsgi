# R3 asset bundle

Large-body receiver fixture used to probe body-size breakpoints with generated payloads.

Contents:
- shared receiver app in `../common/receiver_app.psgi`
- shared corpus and client helpers in `../common/`
- `config.yaml.in` for the scenario
- `r3_matrix.json` for the scenario matrix
- `run_r3_smoke.sh` lightweight smoke runner
